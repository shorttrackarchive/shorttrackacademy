# 쇼트트랙 아카데미 13.3

쇼트트랙 선수를 장기 육성하는 정적 웹게임입니다. 새 선수를 만들고, 매주 3개의 훈련/관리 일정을 선택해 능력치, 컨디션, 피로도, 부상 위험, 관계, 대회 성적, 엔딩 후보를 관리합니다.

## 구조

```text
.
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  └─ app.js
└─ assets/
   └─ images/
```

## GitHub 업로드

1. 이 폴더 전체를 새 GitHub 저장소에 업로드합니다.
2. 저장소 Settings → Pages에서 배포 소스를 `main` 브랜치의 `/root`로 지정합니다.
3. 별도 빌드 명령은 필요 없습니다.

## 수정 반영 사항

- 단일 `index.html` 배포본을 `index.html`, `css/styles.css`, `js/app.js`, `assets/images` 구조로 분리했습니다.
- CSS 분리 후 배경 이미지 경로를 `../assets/images/...`로 보정했습니다.
- 저장 코드/로그/대화/선수 이름 출력에서 HTML 삽입이 실행되지 않도록 이스케이프 처리를 추가했습니다.
- 저장 코드 불러오기 시 최소 게임 데이터 구조를 검증하고 문자열 길이, 배열, 장비 ID 등을 정규화합니다.
- 시즌 랜덤 이벤트 상한에 도달해도 5% 확률로 계속 발생하던 문제를 수정했습니다.
- 시즌 평가 평균 계산이 실제 스탯 개수 기준으로 계산되도록 수정했습니다.
- 커리어 종료 화면에서 같은 제목이 중복 표시되던 부분을 정리했습니다.
